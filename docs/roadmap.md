# Roadmap проекта PW Tactics

Дорожная карта реализации проекта, разбитая по фазам.

### Фаза 1: Прототип (Неделя 1-2)
*Цель: Работающая интерактивная карта с базовой синхронизацией.*
- [x] Развертывание базового проекта (React + Socket.io).
- [x] Отображение карты ГВГ.
- [x] Возможность ставить и передвигать иконку "Катапульта".
- [x] Синхронизация позиции иконки между двумя вкладками браузера.

### Фаза 2: Функциональность тактики (Неделя 3-4)
*Цель: Полноценный инструмент для планирования.*
- [ ] Библиотека иконок с системой Hover (состав пати: иконка+ник) и функционалом выделения (Detachment) юнитов для спец. задач.
- [ ] Расширенный набор инструментов: Ластик, Фигуры (круги, прямоугольники), Текстовые метки.
- [ ] Система Undo/Redo и масштабирование карты (Zoom/Pan).
- [ ] Функционал "Этапов": переключение между различными сценариями (например, "Дракон", "Атака") с изоляцией изменений в реальном времени.
- [ ] Управление доступом: переключатель режимов "Только ведущий" / "Все могут редактировать".

### Фаза 3: Временные шкалы и Интеграция (Неделя 5-6)
*Цель: Динамическое планирование и Docker.*
- [ ] Реализация Timeline: привязка движений и меток ко времени боя.
- [ ] Экспорт и импорт тактики в формате JSON.
- [ ] Упаковка приложения в Docker и настройка docker-compose для VPS.
- [ ] Система автоматической очистки комнат (7 дней неактивности).

### Фаза 4: Релиз и поддержка (Неделя 7+)
*Цель: Публичный доступ и стабильность.*
- [ ] Оптимизация производительности при большом количестве объектов.
- [ ] Деплой на стабильный сервер.
- [ ] Сбор обратной связи от офицеров гильдий и внесение правок.
- [ ] Добавление карт новых ивентов (если появятся).
